# 💰 Finance Dashboard

A modern finance management web application built using React.

## 🚀 Features
- Dashboard with welcome UI
- Add and manage transactions
- Analytics using charts
- Clean and responsive UI

## 🛠 Tech Stack
- React.js
- Vite
- CSS
- Recharts

## ▶️ Run Project
npm install  
npm run dev
